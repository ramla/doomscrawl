# User Guide

## Running
- Clone repo
- `poetry install`
- Run program:
`poetry run invoke start`
- Run the tests, get report:
`poetry run invoke coverage-report`

## Usage
- `Q` to quit
- `WASD` to move character
- `R` to randomise new room (if no room list supplied)
- `T` to initialise triangulation
- `F` to run triangulation all at once
- `E` to run triangulation in steps (also stops running all at once)
